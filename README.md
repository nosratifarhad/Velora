# Velora

Velora is an open-source customer journey automation platform designed to help teams build, orchestrate, and execute event-driven customer journeys.

Inspired by modern customer engagement platforms, Velora enables businesses to create personalized, multi-step workflows triggered by real-time events across multiple communication channels.

## Features

- 🚀 Visual customer journey builder
- ⚡ Event-driven architecture
- 📩 Multi-channel messaging (SMS, Email, Push, WhatsApp, etc.)
- 🎯 Audience segmentation
- 🔀 Workflow automation
- ⏱️ Delays, conditions, and branching
- 📊 Analytics and journey insights
- 🔌 Extensible with plugins and custom integrations
- 🌍 Open-source and community-driven

## Vision

Our mission is to build a modern, scalable, and developer-friendly open-source alternative to proprietary customer engagement platforms.

Whether you're building marketing automation, transactional messaging, onboarding flows, or lifecycle campaigns, Velora provides the foundation for creating intelligent customer journeys.

Contributions are always welcome!
